Fix Common Issues

Solutions for lag, crashes, rendering problems, FPS drops, and more.

🐌

Lag & Stuttering

Timeline playback is choppy and unresponsive
▼

Quick Fixes:

  • Lower playback resolution to 1/4 or 1/8 in the preview window
  • Enable proxy media / optimized media for heavy footage
  • Close background apps to free up RAM (Chrome especially)
  • Move your project files to an SSD drive
  • Clear media cache regularly (Edit > Preferences > Media Cache)

Advanced:

  • Increase allocated RAM in preferences (leave 4GB for OS)
  • Disable "Enable Mercury Transmit" if not using external monitors
  • Use ProRes proxies for 4K+ footage workflows
  • Update GPU drivers to latest version
💥

Crashes & Freezes

Software crashes randomly or on startup
▼

Quick Fixes:

  • Update to the latest GPU drivers (NVIDIA/AMD/Intel)
  • Allocate more RAM to your editor in preferences
  • Disable GPU acceleration if your card is unstable
  • Check for software updates and install latest patches
  • Run as Administrator (Windows) or check permissions (Mac)

Advanced:

  • Delete preferences folder and let software regenerate
  • Check Event Viewer (Windows) or Console (Mac) for error logs
  • Test with a new project to isolate corrupt project files
  • Disable third-party plugins one by one to find conflicts
⏳

Rendering Issues

Export fails, crashes, or produces corrupted files
▼

Quick Fixes:

  • Render to a different codec (try H.264 instead of ProRes)
  • Export to a local drive, not external or network storage
  • Disable GPU rendering and use software-only encoding
  • Break long timelines into shorter segments
  • Check available disk space (need 2x the expected file size)

Advanced:

  • Use Media Encoder instead of direct export from timeline
  • Pre-render effects-heavy sections before final export
  • Match sequence settings to footage to avoid re-encoding
  • Try VBR 1-pass instead of 2-pass if time is critical
📈

FPS Drops & Low Performance

Game recording FPS drops while editing
▼

Quick Fixes:

  • Record at constant bitrate instead of variable bitrate
  • Use hardware encoding (NVENC/AMD VCE/QuickSync)
  • Lower recording resolution if editing on weak hardware
  • Close all background apps while recording
  • Record to a separate drive from your OS and game

Advanced:

  • Use OBS with CQP rate control for consistent quality
  • Cap your in-game FPS to reduce GPU load while recording
  • Allocate more VRAM to recording software
  • Consider a dedicated recording PC (dual-PC setup)
⚡

PC Optimization for Editing

Make your system faster for video editing
▼

Windows Optimization:

  • Set power plan to "High Performance"
  • Disable Windows Game Mode (causes conflicts)
  • Turn off real-time antivirus scanning for project folders
  • Disable visual effects: System > Advanced > Performance
  • Enable hardware-accelerated GPU scheduling

Mac Optimization:

  • Disable automatic graphics switching
  • Clear system cache regularly
  • Use Activity Monitor to find resource hogs
  • Keep at least 20% free space on your drive

Hardware Upgrades (Priority Order):

  • 1. SSD (biggest impact for most users)
  • 2. RAM (32GB minimum for 4K, 64GB for pro work)
  • 3. GPU (8GB+ VRAM for smooth playback)
  • 4. CPU (more cores = faster rendering)
🎧

Audio Issues

No sound, distorted audio, or sync problems
▼

Quick Fixes:

  • Check audio hardware settings and output device
  • Clear audio cache and restart the software
  • Convert variable frame rate audio to constant frame rate
  • Check if audio is muted or solo'd on any track
  • Update audio drivers (Realtek, ASIO, etc.)

Sync Issues:

  • Use "Merge Clips" or "Synchronize" features
  • Manually sync using audio waveforms as reference
  • Record audio and video on same device when possible
  • Use timecode if your devices support it
