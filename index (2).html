Lag & Stuttering

Timeline playback is choppy
▼

Quick Fixes:

  • Lower playback resolution to 1/4 or 1/8
  • Enable proxy media / optimized media
  • Close background apps to free RAM
  • Use an SSD for your project files
  • Clear media cache regularly

Crashes & Freezes

Software keeps crashing
▼

Quick Fixes:

  • Update to the latest GPU drivers
  • Allocate more RAM to your editor
  • Disable GPU acceleration if unstable
  • Check for software updates
  • Run as Administrator (Windows)
